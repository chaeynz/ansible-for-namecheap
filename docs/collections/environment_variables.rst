Index of all Collection Environment Variables
=============================================

The following index documents all environment variables declared by plugins in collections.
Environment variables used by the ansible-core configuration are documented in :ref:`ansible_configuration_settings`.

.. envvar:: NAMECHEAP_API_TOKEN

    The API token created through Namecheap

    *Used by:*
    :ansplugin:`chaeynz.namecheap.nc\_lookup lookup plugin <chaeynz.namecheap.nc_lookup#lookup>`
.. envvar:: NAMECHEAP_TOKEN

    The API token created through Namecheap

    *Used by:*
    :ansplugin:`chaeynz.namecheap.nc\_lookup lookup plugin <chaeynz.namecheap.nc_lookup#lookup>`
.. envvar:: NAMECHEAP_USER

    The username of the Namecheap account

    *Used by:*
    :ansplugin:`chaeynz.namecheap.nc\_lookup lookup plugin <chaeynz.namecheap.nc_lookup#lookup>`
